<h1>Python - Pdf Compressor</h1>


<h2>Description</h2>
Project consists of a simple Python script that enables the user to compress any pdf file whatever the pdf size is. The utility allows you to select any large pdf file that you want to compress with 5 quality options being screen , ebook , printer , prepress and default.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python </b> 
- <b>Command Prompt</b>
- <b>Visual Studio Code</b>

<h2>Libraries Used</h2>

- <b>Tkinter</b>
- <b>Subprocess</b>
- <b>Os</b>


<h2>Environments Used </h2>

- <b>Windows 11</b> (25H2)
